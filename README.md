# Discord-Meme-bot

## Usage: 
This is simple Python code for making Discord Memebot for your own discord servers and other servers.

## Requirements:
Before running this code, you should create your own Discord bot beforehand in official discord developer website.
That way, you can know your discord bot token which will be needed in the code.

## Purpose:
This code's purpose is to connect our code to Discord's backend(More specifically Discord's API).
